# Icon 图标

默认提供一套常用的图标。

## 代码示例

:::demo 基础用法

渲染后形式为`<i class="m-icon m-icon-setting"></i>`的标签。

```html
<template>
    <Icon type="setting" size="24" />
    <Icon type="question-fill" size="24" />
    <Icon type="home" size="24" />
</template>
```
:::


## API

### Icon 图标

| 属性 | 类型 | 说明 | 默认值 |
| ---- | ---- | ---- | ---- |
| type | String | 图标名称 | - |
| size | String, Number | 图标字体大小 | - |
| color | String | 图标字体颜色 | - |
| customIcon | String | 自定义图标名称 | - |
